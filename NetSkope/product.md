# Netskope 產品線

Netskope One Platform 是整合雲端、資料、AI 與網路安全的統一平台，涵蓋 SSE、SD-WAN 與完整 SASE 解決方案。

---

## 一、SSE（Security Service Edge）核心產品

### 1. CASB（Cloud Access Security Broker）

- 管控 SaaS 與雲端應用的存取行為
- 防止資料外洩，支援影子 IT 可視化
- 提供 API 模式與 inline 模式

### 2. Next-Gen SWG（Secure Web Gateway）

- 下一代安全網頁閘道，過濾惡意網路流量
- 單一通道架構（single-pass）檢查 web 與雲端流量
- 支援 SSL/TLS 解密與威脅防護

### 3. ZTNA / Private Access（Zero Trust Network Access）

- 取代傳統 VPN，以零信任原則控管私有應用存取
- 依據使用者身份、裝置狀態與情境動態授權
- 支援遠端工作者與分支機構

### 4. Cloud Firewall

- 雲端原生防火牆，保護出站流量
- 集中管理網路存取政策

### 5. DLP（Data Loss Prevention）

- 跨所有向量（Web、雲端、端點）的資料外洩防護
- 支援結構化與非結構化資料識別
- 整合 CASB、SWG、ZTNA 一致執行政策

### 6. UEBA（User & Entity Behavior Analytics）

- 偵測異常使用者行為與內部威脅
- 透過機器學習建立行為基準線

### 7. SkopeAI

- AI 驅動的威脅偵測、資料分類與風險評估
- 支援生成式 AI 應用的安全管控（如 ChatGPT、Copilot）

---

## 二、SD-WAN 產品

### 1. Netskope SD-WAN

- 提供安全、高效能的廣域網路連線
- 支援分支機構、遠端用戶與雲端應用

### 2. Netskope Borderless SD-WAN

- 整合安全功能的 SD-WAN 方案
- 與 SSE 深度整合，實現 SASE 架構

---

## 三、整合平台

### Netskope One SSE

- 將 CASB、SWG、ZTNA、DLP、Firewall 整合為單一 SSE 平台
- Gartner Magic Quadrant SSE 領導者（2022 年起連續入選）

### Netskope One SASE

- SSE + SD-WAN 的完整 SASE 解決方案
- 統一管理主控台，單一平台涵蓋所有產品
